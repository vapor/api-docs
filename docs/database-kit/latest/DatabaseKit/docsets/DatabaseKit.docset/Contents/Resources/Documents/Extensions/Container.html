Container

protocol Container : BasicWorker

Containers that have DatabasesConfig structs registered can be used to open, pool, and cache connections.

  • requestCachedConnection(to:)

    Returns a Future connection to the Database specified by the supplied DatabaseIdentifier. Subsequent calls to this method with the same database ID will return the same connection. You must call releaseCachedConnections() to release the connections.

    let conn = try app.requestCachedConnection(to: .psql).wait()
// use conn
try app.releaseCachedConnections()

New
  • withNewConnection(to:closure:)

    Creates a new connection to the Database specified by the supplied DatabaseIdentifier.

    The connection is provided to the supplied callback and will be automatically closed when the future returned by the callback is completed.

    app.withNewConnection(to: .psql) { conn in
    // use the connection
}

    See newConnection(...) to create a connection without using a callback.
