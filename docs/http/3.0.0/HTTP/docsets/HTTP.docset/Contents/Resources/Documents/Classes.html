  • HTTPChunkedStream

    A "Transfer-Encoding: chunked" stream of data used by HTTPBody.

    let chunkedStream = HTTPChunkedStream(on: req)
background {
    while true {
        sleep(1)
        try! chunkedStream.write(...).wait()
    }
}
return try HTTPResponse(status: .ok, body: chunkedStream)

    HTTPChunkedStream allows you to send data asynchronously without a predefined length. The HTTPMessage will be considered complete when the end chunk is sent.
