Allows HTTPServer to accept WebSocket connections.

    // return non-nil HTTPHeaders to allow upgrade
}, onUpgrade: { ws, req in

})

    maxFrameSize

    Maximum WebSocket frame size this server will accept.

    Called when an incoming HTTPRequest attempts to upgrade. Return non-nil headers to accept the upgrade.
