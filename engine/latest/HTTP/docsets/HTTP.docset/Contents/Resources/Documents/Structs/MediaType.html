MediaType

public struct MediaType: Hashable

A media type (also MIME type and content type) is a two-part identifier for file formats and format contents transmitted on the Internet.

https://en.wikipedia.org/wiki/Media_type https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Content-Type
